# 🌐 Praveena's Personal Portfolio

A modern, responsive personal portfolio website showcasing my skills, projects, and experience in **Data Science**, **Machine Learning**, **Artificial Intelligence**, and **Software Development**.

🔗 **Live Website:** https://portfolio-alpha-opal-81.vercel.app/

---

## 🚀 Features

- ✨ Modern and responsive UI
- 📱 Mobile-friendly design
- 🎬 Smooth GSAP animations
- 🌌 Interactive AI-inspired animated background
- 🖱️ Smooth scrolling navigation
- 📂 Dedicated projects showcase
- 📄 Resume download
- 📬 Working contact form powered by Formspree
- 🔍 SEO optimized
- ⚡ Fast deployment with Vercel
